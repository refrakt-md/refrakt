# @refrakt-md/places

Event planning and location runes for travel guides, venue pages, and conference sites. Structure events with agendas, embed interactive maps, and build day-by-day itineraries — all from Markdown.

## Installation

```bash
npm install @refrakt-md/places
```

```json
{
  "packages": ["@refrakt-md/places"]
}
```

## Runes

| Rune | Description |
|------|-------------|
| [event](/docs/runes/places/event) | Event information with date, location, and agenda |
| [map](/docs/runes/places/map) | Interactive map visualization from Markdown lists of locations |
| [itinerary](/docs/runes/places/itinerary) | Day-by-day travel itineraries with timed stops and locations |

## When to use

Use this package for travel blogs, conference sites, city guides, or any content that revolves around places and events. Combine `event` for scheduled happenings, `map` for spatial context, and `itinerary` for multi-day trip planning.
