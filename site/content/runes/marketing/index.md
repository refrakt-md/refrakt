# @refrakt-md/marketing

Landing page and conversion runes for marketing sites and product pages. Build hero sections, feature grids, pricing tables, testimonials, and calls to action — all from Markdown.

## Installation

```bash
npm install @refrakt-md/marketing
```

```json
{
  "packages": ["@refrakt-md/marketing"]
}
```

## Runes

| Rune | Description |
|------|-------------|
| [hero](/runes/marketing/hero) | Full-width intro sections with background support and action buttons |
| [cta](/runes/marketing/cta) | Focused call-to-action blocks with headlines and action buttons |
| [bento](/runes/marketing/bento) | Magazine-style bento grid where heading levels control cell size |
| [feature](/runes/marketing/feature) | Feature showcases with name, description, and optional icons |
| [steps](/runes/marketing/steps) | Step-by-step instructions with numbered indicators |
| [pricing](/runes/marketing/pricing) | Pricing tables with tier comparison |
| [testimonial](/runes/marketing/testimonial) | Customer testimonials and reviews |
| [comparison](/runes/marketing/comparison) | Product and feature comparison matrices from Markdown |

## When to use

Use this package when building marketing sites, product landing pages, or any page that needs conversion-oriented content blocks. The runes are designed to work together — a typical landing page might combine a `hero` at the top, `feature` and `steps` in the middle, `testimonial` for social proof, `pricing` for plans, and a `cta` at the bottom.
