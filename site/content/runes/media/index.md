# @refrakt-md/media

Audio and music runes for playlists, tracks, and audio players. Build album pages, podcast listings, and audio content — all from Markdown.

## Installation

```bash
npm install @refrakt-md/media
```

```json
{
  "packages": ["@refrakt-md/media"]
}
```

## Runes

| Rune | Description |
|------|-------------|
| [playlist](/runes/media/playlist) | Curated playlist with track listing for albums, podcasts, and mixes |
| [track](/runes/media/track) | Standalone track or recording with metadata |
| [audio](/runes/media/audio) | Audio player with optional waveform and chapter markers |

## When to use

Use this package for music blogs, podcast sites, audio portfolios, or any content featuring audio media. The `playlist` rune organizes collections of tracks, `track` handles individual recordings with metadata, and `audio` provides a player with waveform visualization.
