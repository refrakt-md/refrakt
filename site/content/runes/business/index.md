# @refrakt-md/business

Organizational runes for team directories, company profiles, and timelines. Build people pages, org structures, and chronological histories — all from Markdown.

## Installation

```bash
npm install @refrakt-md/business
```

```json
{
  "packages": ["@refrakt-md/business"]
}
```

## Runes

| Rune | Description |
|------|-------------|
| [cast](/runes/business/cast) | People directory for team pages and speaker lineups |
| [organization](/runes/business/organization) | Structured business or organization information |
| [timeline](/runes/business/timeline) | Chronological events displayed as a timeline |

## When to use

Use this package for company websites, team pages, about sections, and organizational content. The `cast` rune handles team member grids and speaker lineups, `organization` structures company info, and `timeline` presents milestones and history chronologically.
