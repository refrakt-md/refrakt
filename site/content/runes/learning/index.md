# @refrakt-md/learning

Educational runes for tutorials, recipes, and instructional content. Write structured how-to guides and cooking recipes with semantic markup that supports rich snippets and SEO.

## Installation

```bash
npm install @refrakt-md/learning
```

```json
{
  "packages": ["@refrakt-md/learning"]
}
```

## Runes

| Rune | Description |
|------|-------------|
| [howto](/runes/learning/howto) | Step-by-step how-to guide with tools and instructions |
| [recipe](/runes/learning/recipe) | Structured recipe with ingredients, steps, and chef tips |

## When to use

Use this package for tutorial sites, cooking blogs, DIY guides, or any instructional content. Both runes produce structured data (JSON-LD) that search engines use for rich snippets — how-to cards and recipe cards in search results.
