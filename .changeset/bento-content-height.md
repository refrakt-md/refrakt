Add an opt-in `content-height` attribute to the `bento` rune (`sm` | `md` | `lg`)
that pins each cell's text area to a fixed height (3 / 5 / 7rem) so cells line up
vertically in grid mode — the media zone absorbs the remaining row-track height.
Defaults to natural height (unchanged), and reverts to natural height on the
mobile stack.
