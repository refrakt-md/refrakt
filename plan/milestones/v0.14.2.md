# v0.14.2 — Curated syntax preset lineup (Phase 1)

The v0.14.2 milestone implements {% ref "SPEC-057" /%} — imports six widely-recognised syntax palettes (**Dracula, Solarized, Catppuccin, Tokyo Night, One Dark, Gruvbox**) as first-party refrakt presets, following the pattern established by Nord in {% ref "WORK-220" /%}.

Application work, not architecture. Every preset uses the existing SPEC-056 mechanism unchanged: a `ThemeTokensConfig` module under `packages/lumina/src/presets/`, a tint registration in the docs site config, and a doc page following the structure validated on Nord. The architecture is stable; the cost here is volume + palette-mapping accuracy.

A patch milestone scoped to a single spec. Builds on {% ref "SPEC-056" /%} (token contract extension), {% ref "SPEC-053" /%} (tint vocabulary), and {% ref "SPEC-048" /%} (token contract foundation).

## Implementation order

Every preset work item is independently parallelisable — each ships a module, a test file, and a doc page in one PR. They can be implemented in any order. WORK-230 (catalog wrap-up) lands last because it touches the themes-catalog landing page and the `tints` config in one go, after all six presets exist.

Suggested order (peak name recognition first, so we land the most-visible wins early):

1. **{% ref "WORK-224" /%}** — Dracula (dark-only, name recognition)
2. **{% ref "WORK-225" /%}** — Solarized (light + dark from same 16 hues — validates unified-mode shape)
3. **{% ref "WORK-226" /%}** — Catppuccin (Latte + Mocha — modern, well-spec'd)
4. **{% ref "WORK-227" /%}** — Tokyo Night (Day + Storm — exercises SPEC-056 extended roles maximally)
5. **{% ref "WORK-228" /%}** — One Dark (Atom's signature, dark-only Phase 1)
6. **{% ref "WORK-229" /%}** — Gruvbox (light + dark, warm counterweight)
7. **{% ref "WORK-230" /%}** — Catalog wrap-up (tints config, themes-catalog landing copy, layout nav)

## Out of milestone (deferred)

- **Phase 2 palettes** — GitHub Light/Dark, Monokai, Ayu, Rosé Pine, Kanagawa. Easy to add once the import pattern is proven across six palettes; defer to a future milestone if there's demand.
- **Additional flavours** of multi-flavour palettes — Catppuccin Frappé/Macchiato, Tokyo Night Moon. The canonical pair (Latte+Mocha, Day+Storm) is sufficient for Phase 1.
- **Per-palette npm packages** — palettes stay inside `@refrakt-md/lumina/presets/`. Splitting into per-palette plugin packages is a packaging decision deferred indefinitely.
- **Architectural changes** — SPEC-057 forbids any changes to `SyntaxTokens`, `TintDefinition`, the scope-eligibility filter, the extended Shiki theme, or the generator. If a palette doesn't fit cleanly, the palette is deferred — not the contract revised.
